== Description ==

WooCommerce外贸跨境物流价格查询插件是一个小巧，但强大的物流插件。安装后，可以经过简单配置，就可以在用户下单时，看到相关估价，这样客户在付款时对运费进行付款。我们支持顺丰国际小包挂号、顺丰国际电商专递CD、E特快-MINHOUYOUZHENG、E邮宝-MINHOUYOUZHENG、云途全球专线挂号（带电）、云途全球专线挂号(特惠普货)、美邮宝（加拿大专线）、天梯中美专线特惠（USPS）、DHL、上海守务（美国小包普货-特快JX）、莆田4PX-普货专线、莆田4PX-口罩专线、燕文物流(追踪普货)欧洲、燕文物流(追踪普货)北美、燕文专线追踪普货(鞋服专线)、义达物流全球快捷-普货、义达物流(加拿大）、义达物流全球快捷-普货-AZGLON、义达物流(日本)、义达物流(日本-带电)。本插件提供的KEY，在 http://www.wuliuapi.qifeiye.com/ 可以获取，每月2000次，超过以后，联系我们。由此引发的问题与我们无关，我们不承担任何责任，也不会进行任何赔偿。本插件只是一个物流估价插件，不是物流跟踪，如需物流跟踪，请绕行。

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. In WooCommerce store Settings, add delivery methods and define discounts to check logistics costs in the front end
